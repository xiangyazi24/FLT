# Q2381 drop: `E1 : Y^2 = X(X-1)(X+3)` rational-point certificate roadmap

Target affine curve over `ℚ`:

```text
E1 : Y^2 = X^3 + 2 X^2 - 3 X = X * (X - 1) * (X + 3).
```

Expected affine rational points:

```text
(-3,0), (0,0), (1,0), (-1,2), (-1,-2), (3,6), (3,-6).
```

The previous `E24` shift is

```text
X = U - 1,  Y = V,
E24 : V^2 = U^3 - U^2 - 4U + 4.
```

So the corresponding `E24` affine list is

```text
(-2,0), (1,0), (2,0), (0,2), (0,-2), (4,6), (4,-6).
```

## Executive recommendation

Use a **2-isogeny descent plus a mod-5 torsion bound** as the primary Lean certificate.  This is much smaller and safer than trying to solve every full 2-covering intersection of quadrics directly.

The trusted mathematical residuals can be reduced to standard named elliptic-curve facts:

1. the exactness of the 2-isogeny descent maps for a curve `y^2 = x^3 + a*x^2 + b*x`,
2. Mordell-Weil finite generation / the formula `#(E(ℚ)/2E(ℚ)) = 2^rank * #(E[2](ℚ))`,
3. injectivity of torsion reduction at a prime of good reduction.

Everything else below is pure rational/integer arithmetic: formula checking, quartic-cover case checking, expected-point evaluation, finite-field counting over `F_5`, and the `E24` shift.

The key point is adversarial: **do not try to conclude the affine point list merely from local solubility of the `α`-covers.**  The four `α`-cover squareclasses all survive, and solving those covers globally is essentially the original rational-point problem.  The rank-zero step plus the mod-5 torsion bound is what closes the list.

## 1. Curves and exact 2-isogeny data

Put

```text
E  : y^2 = x^3 + 2*x^2 - 3*x = x*(x-1)*(x+3),
E' : Y^2 = X^3 - 4*X^2 + 16*X = X*(X^2 - 4*X + 16).
```

This is the standard 2-isogeny from

```text
y^2 = x^3 + a*x^2 + b*x
```

with `(a,b) = (2,-3)` and kernel generated by `(0,0)`.  The 2-isogenous curve is

```text
y^2 = x^3 - 2*a*x^2 + (a^2 - 4*b)*x
    = x^3 - 4*x^2 + 16*x.
```

The isogeny `φ : E → E'` is

```text
φ(O)       = O,
φ((0,0))   = O,
φ(x,y)     = ( y^2 / x^2,
               y * (b - x^2) / x^2 )
           = ( y^2 / x^2,
              -y * (x^2 + 3) / x^2 )       for x ≠ 0.
```

The dual isogeny `ψ : E' → E` is

```text
ψ(O)       = O,
ψ((0,0))   = O,
ψ(X,Y)     = ( Y^2 / (4*X^2),
               Y * (16 - X^2) / (8*X^2) )  for X ≠ 0.
```

The factor `4,8` in the dual is not optional: applying the same kernel formula to `E'` first lands on the quadratic twist/scaling

```text
y^2 = x^3 + 8*x^2 - 48*x,
```

and then the scaling `(x,y) ↦ (x/4,y/8)` returns to `E`.

### 2.1 `α` on `E`

For `E : y^2 = x^3 + 2*x^2 - 3*x`, define

```text
α_E(O)       = 1,
α_E((0,0))   = b = -3,
α_E((x,y))   = x mod (ℚ*)^2        for x ≠ 0.
```

The associated affine quartic cover for a squareclass representative `d` is

```text
Cα(d) : d*w^2 = d^2*z^4 + 2*d*z^2 - 3.
```

The map from a cover point to `E` is

```text
x = d*z^2,
y = d*z*w.
```

Indeed,

```text
y^2 = d^2*z^2*w^2
    = d*z^2 * (d*w^2)
    = d*z^2 * (d^2*z^4 + 2*d*z^2 - 3)
    = x * (x^2 + 2*x - 3).
```

For actual `α_E(E(ℚ))`, the only squareclasses needed are

```text
d ∈ { 1, -1, 3, -3 }.
```

This is the usual `b`-divisor support result for the kernel-`(0,0)` 2-isogeny: here `b = -3`.  If no EC descent API is available, keep this as part of the named 2-isogeny descent residual rather than reproving valuations ad hoc.

Case table for `Cα`:

| `d` | normalized cover equation over `ℚ` | status / witness | point(s) on `E` obtained from `x=d*z^2`, `y=d*z*w` |
|---:|---|---|---|
| `1` | `w^2 = z^4 + 2*z^2 - 3` | soluble, `z=1,w=0`; also the projective point at infinity is the identity convention | `(1,0)` and `O` |
| `-3` | `w^2 = -3*z^4 + 2*z^2 + 1` | soluble, `z=0,w=±1` and `z=1,w=0` | `(0,0)` and `(-3,0)` |
| `-1` | `w^2 = -z^4 + 2*z^2 + 3` | soluble, `z=1,w=±2` | `(-1,-2)`, `(-1,2)` |
| `3` | `w^2 = 3*z^4 + 2*z^2 - 1` | soluble, `z=1,w=±2` | `(3,6)`, `(3,-6)` |

Thus

```text
α_E(E(ℚ)) = { 1, -1, 3, -3 }
```

as squareclasses.  The inclusion `⊇` is just the witness table above.  The inclusion `⊆` is the standard support/descent statement.

### 2.2 `β` on `E'`

For

```text
E' : Y^2 = X^3 - 4*X^2 + 16*X,
```

define

```text
β_E'(O)       = 1,
β_E'((0,0))   = 16 = 1 mod squares,
β_E'((X,Y))   = X mod (ℚ*)^2        for X ≠ 0.
```

The associated quartic cover is

```text
Cβ(d) : d*w^2 = d^2*z^4 - 4*d*z^2 + 16,
```

with map

```text
X = d*z^2,
Y = d*z*w.
```

The only squareclasses needed for `β_E'` are squarefree divisors of `16`, including sign:

```text
d ∈ { 1, -1, 2, -2 }.
```

Case table for `Cβ`:

| `d` | normalized equation | status |
|---:|---|---|
| `1` | `w^2 = z^4 - 4*z^2 + 16` | soluble, e.g. `z=0,w=±4` gives `(0,0)`; `z=2,w=±4` gives `(4,±8)` |
| `-1` | `w^2 = -z^4 - 4*z^2 - 16` | impossible over `ℚ` by positivity of `z^4 + 4*z^2 + 16` |
| `-2` | `w^2 = -2*z^4 - 4*z^2 - 8` | impossible over `ℚ` by positivity of `2*z^4 + 4*z^2 + 8` |
| `2` | `w^2 = 2*z^4 - 4*z^2 + 8` | impossible over `ℚ` by the mod-16 integer lemma below |

Therefore

```text
β_E'(E'(ℚ)) = { 1 }.
```

The `d=2` contradiction is very small and should be coded over `Int`.

Suppose `z=m/n` with `gcd(m,n)=1`, and set `k = w*n^2`, which is an integer after clearing denominators.  Then

```text
k^2 = 2*m^4 - 4*m^2*n^2 + 8*n^4.
```

Modulo `16`:

* if `m,n` are both odd, RHS is `2 - 4 + 8 = 6 mod 16`,
* if `m` is odd and `n` is even, RHS is `2 mod 16`,
* if `m` is even and `n` is odd, RHS is `8 mod 16`.

The remaining parity case `m,n` both even is excluded by `gcd(m,n)=1`.  But an integer square modulo `16` is only `0,1,4,9`.  Contradiction.

## 3. Descent exactness and rank-zero theorem DAG

The standard exact descent facts give

```text
E(ℚ) / ψ(E'(ℚ))  ≅ α_E(E(ℚ)),
E'(ℚ) / φ(E(ℚ)) ≅ β_E'(E'(ℚ)).
```

From the case tables:

```text
# α_E(E(ℚ))  = 4,
# β_E'(E'(ℚ)) = 1.
```

Hence

```text
# E(ℚ) / 2E(ℚ)
  = # E(ℚ) / ψ(E'(ℚ)) * # E'(ℚ) / φ(E(ℚ))
  = 4 * 1
  = 4.
```

But `E` has full rational 2-torsion:

```text
E[2](ℚ) = { O, (0,0), (1,0), (-3,0) },
#E[2](ℚ) = 4.
```

For a finitely generated abelian Mordell-Weil group,

```text
# E(ℚ)/2E(ℚ) = 2^rank * #E[2](ℚ).
```

Therefore

```text
rank E(ℚ) = 0.
```

## 4. Torsion closure by reduction modulo 5

The discriminant of `E : y^2 = x(x-1)(x+3)` is

```text
Δ = 16 * (0-1)^2 * (0+3)^2 * (1+3)^2
  = 16 * 1 * 9 * 16
  = 2304 = 2^8 * 3^2.
```

Thus `p=5` is a prime of good reduction.

Count `E(F_5)` directly:

```text
f(x) = x*(x-1)*(x+3) over F_5.

x = 0 : f=0, one y
x = 1 : f=0, one y
x = 2 : f=0, one y      -- because 2 = -3 mod 5
x = 3 : f=1, two y = ±1
x = 4 : f=4, two y = ±2
plus O.
```

So

```text
#E(F_5) = 1 + 1 + 1 + 1 + 2 + 2 = 8.
```

Since torsion reduction is injective at a good prime,

```text
#E(ℚ)_tors ≤ #E(F_5) = 8.
```

From rank zero, every rational point is torsion.  The following eight projective rational points are visibly on `E`:

```text
O,
(-3,0), (0,0), (1,0),
(-1,2), (-1,-2),
(3,6), (3,-6).
```

Therefore there are exactly these eight projective rational points, and exactly the seven listed affine rational points.

## 6. Optional full 2-covering cross-check

This section is **not** the recommended primary proof, but it is useful to prevent hidden squareclass mistakes.

For `Y ≠ 0`, all three factors are nonzero:

```text
X, X-1, X+3 ≠ 0.
```

Use squareclass representatives

```text
D = { ±1, ±2, ±3, ±6 }.
```

Choose integers/rationals `A,B,C,T` with `T ≠ 0` such that

```text
X     = d0 * A^2 / T^2,
X - 1 = d1 * B^2 / T^2,
X + 3 = d3 * C^2 / T^2.
```

Then

```text
FC1 : d0*A^2 - d1*B^2 = T^2,
FC2 : d3*C^2 - d0*A^2 = 3*T^2,
FC3 : d0*d1*d3 is a positive rational squareclass.
```

Equivalently, for the chosen squarefree representatives, `d0*d1*d3` must be one of

```text
1, 4, 9, 36.
```

The real sign order of the factors gives only two sign patterns for nonzero `Y`:

```text
X > 1        : signs +,+,+,
-3 < X < 0   : signs -,-,+.
```

The sign patterns `+,-,-` and `-,+,-` are impossible from the ordering of `X`, `X-1`, `X+3`; the patterns with negative product are impossible from `Y^2`.

The two actual non-torsion-looking survivor triples are

```text
(-1,-2, 2)  with A=B=C=T=1, giving X=-1 and Y=±2,
( 3, 2, 6)  with A=B=C=T=1, giving X= 3 and Y=±6.
```

Indeed:

```text
-1 - (-2) = 1,       2 - (-1) = 3,
 3 -   2  = 1,       6 -   3  = 3.
```

And

```text
(-1)*(-2)*2 = 4,     so |Y| = 2,
3*2*6       = 36,    so |Y| = 6.
```

The full sign-allowed squareclass triples before finite elimination are:

```text
Positive pattern +++:
(1,1,1), (1,2,2), (1,3,3), (1,6,6),
(2,1,2), (2,2,1), (2,3,6), (2,6,3),
(3,1,3), (3,2,6), (3,3,1), (3,6,2),
(6,1,6), (6,2,3), (6,3,2), (6,6,1).

Negative-negative-positive pattern --+:
(-1,-1,1), (-1,-2,2), (-1,-3,3), (-1,-6,6),
(-2,-1,2), (-2,-2,1), (-2,-3,6), (-2,-6,3),
(-3,-1,3), (-3,-2,6), (-3,-3,1), (-3,-6,2),
(-6,-1,6), (-6,-2,3), (-6,-3,2), (-6,-6,1).
```

If you want a direct full-2-covering proof instead of the 2-isogeny/torsion route, the honest residual should be exactly this:

