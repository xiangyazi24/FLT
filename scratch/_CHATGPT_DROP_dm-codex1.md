# Q2208 response: finite 2-isogeny/Selmer certificate for the shifted N=12 curve

Shifted curve:

```text
F  : Y^2 = X^3 + 2 X^2 - 3 X = X (X - 1) (X + 3)
F' : Y^2 = X^3 - 4 X^2 + 16 X = X (X^2 - 4 X + 16)
```

The intended 2-isogeny is the one with kernel generated by `(0,0)` on `F`. The audit result is:

```text
im(alpha_F)  = {1, -1, 3, -3}
im(alpha_F') = {1}
```

This is correct, with two important caveats:

1. `alpha_F((0,0))` is not `0`; by the standard convention it is the class of `b = -3`.
2. `alpha_F'((0,0))` is the class of `b' = 16`, hence the trivial squareclass `1`.

With these conventions, the isogeny descent formula gives

```text
2^rank(F(Q)) = (# im(alpha_F) * # im(alpha_F')) / #F(Q)[2]
             = (4 * 1) / 4
             = 1,
```

so `rank F(Q) = 0`.

## 1. Descent maps

Use the standard model

```text
G_{a,b} : y^2 = x^3 + a x^2 + b x = x (x^2 + a x + b),
```

with rational 2-torsion point `(0,0)`. The associated 2-isogenous curve is

```text
G' : y^2 = x^3 - 2a x^2 + (a^2 - 4b) x.
```

For `F`, `a = 2`, `b = -3`, so

```text
F' : Y^2 = X^3 - 4 X^2 + 16 X.
```

The Kummer/descent map attached to the dual isogeny is

```text
alpha_G : G(Q) -> Q*/Q*2
alpha_G(O)       = 1,
alpha_G((0,0))   = b,
alpha_G((x,y))   = x       if x != 0.
```

For `F`, this is

```text
alpha_F(O)       = 1,
alpha_F((0,0))   = -3,
alpha_F((X,Y))   = X       if X != 0.
```

For the dual curve `F'`, `a' = -4`, `b' = 16`, and

```text
alpha_F'(O)       = 1,
alpha_F'((0,0))   = 16 = 1 in Q*/Q*2,
alpha_F'((X,Y))   = X        if X != 0.
```

The kernel statements to use in the rank formula are:

```text
ker(alpha_F)  = dual_isogeny(F'(Q)),
ker(alpha_F') = isogeny(F(Q)).
```

Thus

```text
#im(alpha_F)  = # F(Q)  / dual_isogeny(F'(Q)),
#im(alpha_F') = # F'(Q) / isogeny(F(Q)).
```

Since `F(Q)[2]` has the four points

```text
O, (-3,0), (0,0), (1,0),
```

one obtains

```text
#im(alpha_F) * #im(alpha_F') = #F(Q)/2F(Q) = 2^rank * 4.
```

## 2. Homogeneous spaces

For `G_{a,b}`, a squareclass `d` is represented by the quartic homogeneous space

```text
C_d : W^2 = d U^4 + a U^2 V^2 + (b/d) V^4,
```

where `d` is chosen so that `b/d` is integral in the finite candidate list. If `U != 0`, the corresponding affine point is recovered by

```text
x = d (U/V)^2,
y = d U W / V^3.
```

The special points `O` and `(0,0)` are handled by the conventions for `alpha` above; the projective quartic still gives the correct finite certificate.

### 2.1 Candidate support for `alpha_F`

For `F`, `b = -3`. The elementary support lemma is:

```text
alpha_F(P) is represented by one of 1, -1, 3, -3.
```

A Lean-friendly proof is by writing a nonzero `X`-coordinate in primitive square-denominator form

```text
X = A / B^2,
Y = C / B^3,
gcd(A,B)=1.
```

Then

```text
C^2 = A (A^2 + 2 A B^2 - 3 B^4),
```

and

```text
gcd(A, A^2 + 2 A B^2 - 3 B^4) | 3.
```

Therefore every prime appearing to odd valuation in `A` is `3`; the sign is arbitrary. Hence the squareclass universe for `alpha_F` is exactly

```text
D_F = {1, -1, 3, -3}.
```

All four classes have rational witnesses:

| class `d` | quartic equation | point on `F` | witness `(U,V,W)` |
|---:|---|---:|---:|
| `1` | `W^2 = U^4 + 2U^2V^2 - 3V^4` | `(1,0)` | `(1,1,0)` |
| `-1` | `W^2 = -U^4 + 2U^2V^2 + 3V^4` | `(-1,2)` | `(1,1,-2)` |
| `3` | `W^2 = 3U^4 + 2U^2V^2 - V^4` | `(3,6)` | `(1,1,2)` |
| `-3` | `W^2 = -3U^4 + 2U^2V^2 + V^4` | `(-3,0)` | `(1,1,0)` |

So the exact image is

```text
im(alpha_F) = {1, -1, 3, -3}.
```

No local rejection certificate is needed on this side: every possible class is visibly globally soluble.

### 2.2 Candidate support for `alpha_F'`

For `F'`, `b' = 16`. The same primitive denominator argument gives

```text
alpha_F'(P) is represented by one of 1, -1, 2, -2.
```

Indeed, if

```text
X = A / B^2,
Y = C / B^3,
gcd(A,B)=1,
```

then

```text
C^2 = A (A^2 - 4 A B^2 + 16 B^4),
gcd(A, A^2 - 4 A B^2 + 16 B^4) | 16.
```

Thus only the prime `2` can appear to odd valuation in the squareclass of `A`. The candidate universe for `alpha_F'` is

```text
D_F' = {1, -1, 2, -2}.
```

The four homogeneous spaces are:

| class `d` | equation | status |
|---:|---|---|
| `1` | `W^2 = U^4 - 4U^2V^2 + 16V^4` | survives; witness `(U,V,W)=(2,1,4)`, giving `(X,Y)=(4,8)` |
| `-1` | `W^2 = -U^4 - 4U^2V^2 - 16V^4` | rejected over `R` |
| `2` | `W^2 = 2U^4 - 4U^2V^2 + 8V^4` | rejected over `Q_2`, already modulo `16` |
| `-2` | `W^2 = -2U^4 - 4U^2V^2 - 8V^4` | rejected over `R` |

For the negative classes, the right side is strictly negative for `(U,V) != (0,0)`, while `W^2 >= 0`. This is the most Lean-friendly real obstruction.

For the class `2`, use the primitive integral equation

```text
W^2 = 2U^4 - 4U^2V^2 + 8V^4,
gcd(U,V)=1.
```

Modulo `16`:

* if `U` is odd and `V` is even, the right side is `2 mod 16`;
* if `U` and `V` are both odd, the right side is `6 mod 16`;
* if `U` is even and `V` is odd, the right side is `8 mod 16`.

The coprimality condition excludes both `U,V` even, and no square modulo `16` is `2`, `6`, or `8`. Hence the class `2` has no `Q_2` point.

Therefore

```text
im(alpha_F') = {1}.
```

Again, this is not just a Selmer upper bound: the only surviving class has a rational witness.

## 4. The rank-zero and point-list boundary theorems

The existing N=12 development should not depend on the internal details of a future descent formalization. It should depend on one of the following boundary theorems.

### Boundary theorem A: rank zero from the isogeny certificate

```lean
/-- Hard arithmetic boundary: the finite 2-isogeny certificate proves rank zero. -/
theorem F_rank_zero_from_2_isogeny_certificate :
    -- mathematically: rank F(Q) = 0
    True := by
  -- Uses:
  -- 1. alphaF_image_certificate:  #im alpha_F = 4
  -- 2. alphaFp_image_certificate: #im alpha_F' = 1
  -- 3. standard 2-isogeny descent card/rank formula
  --      #im alpha_F * #im alpha_F' = 2^rank * #F(Q)[2]
  -- 4. #F(Q)[2] = 4
  trivial
```

This theorem still requires the standard global 2-isogeny descent theorem. The finite parts above are elementary certificates; the exact sequence and rank formula are the hard general arithmetic ingredient.

### Boundary theorem B: exact affine rational point list for `F`

```lean

    {X Y : Rat}
    (hF : Y^2 = X^3 + 2 * X^2 - 3 * X) :
    (X = -3 and Y = 0) or
    (X = 0 and Y = 0) or
    (X = 1 and Y = 0) or
    (X = -1 and (Y = 2 or Y = -2)) or
    (X = 3 and (Y = 6 or Y = -6)) := by
  -- rank zero + torsion enumeration
  sorry
```

To derive this from rank zero, use the finite torsion certificate:

```text
#F(F_5) = 8,
#F(F_7) = 8,
```

with good reduction at `5` and `7`. Since torsion injects into good reduction at good primes,

```text
#F(Q)_tors | gcd(8,8) = 8.
```

The following eight rational points are visible:

```text
O,
(-3,0), (0,0), (1,0),
(-1,2), (-1,-2),
(3,6), (3,-6).
```

Rank zero then implies this is the complete rational point set.

The point counts are finite and should be formalized by direct enumeration over `ZMod 5` and `ZMod 7`:

### Boundary theorem C: original obstruction curve point list

The original curve is

```text
E : y^2 = x^3 - x^2 - 4x + 4.
```

It is the shift `X = x - 1`, `Y = y`. Thus `F_affine_rational_points` immediately gives


## 5. Elementary wrappers that can already be separated

Once `E_affine_rational_points` is available, the denominator residual is elementary.

```lean
/-- Primitive square-denominator model on E has denominator one. -/
theorem E_square_denominator_forces_B_one
    {A B C : Int}
    (hBpos : 0 < B)
    (hcop : Int.gcd A B = 1)
    (hEq : C^2 = (A - B^2) * (A - 2 * B^2) * (A + 2 * B^2)) :
    B = 1 := by
  -- Let x = A / B^2 and y = C / B^3.
  -- Then `(x,y)` lies on E.
  -- The point-list theorem gives x in {-2,0,1,2,4}.
  -- Hence A = x * B^2, so B | A.
  -- With gcd(A,B)=1 and B>0, conclude B=1.
  sorry

/-- Target residual contradiction. -/
theorem N12_target_residual_false
    {A B C : Int}
    (hB : B > 1)
    (hcop : Int.gcd A B = 1)
    (hEq : C^2 = (A - B^2) * (A - 2 * B^2) * (A + 2 * B^2)) :
    False := by
  have hBpos : 0 < B := by omega
  have hB1 := E_square_denominator_forces_B_one
    (A := A) (B := B) (C := C) hBpos hcop hEq
  omega
```

The following sublemmas are also elementary wrappers and do not need the isogeny-descent machinery:

```text
1. residual equation -> rational point on E;
2. shift from E to F by X=x-1;
3. primitive denominator algebra: if A/B^2 is an integer and gcd(A,B)=1, then B=1;
4. support lemmas for alpha_F and alpha_F';
5. the mod-16 rejection of the dual class 2;
6. the real rejection of the dual negative classes;
7. finite point counts over ZMod 5 and ZMod 7.
```

The non-elementary boundary remains:

```text
standard 2-isogeny descent exact sequence/cardinality formula
+ torsion injection into good reduction, unless already available in Mathlib/project code.
```

## 6. False shortcuts and missing hypotheses to avoid

1. **Do not define `alpha((0,0)) = 0`.** The target is `Q*/Q*2`, so `0` is not allowed. The special value is `b` for `G_{a,b}`. For `F`, this is `-3`; for `F'`, it is `16`, hence `1`.

2. **Do not claim `im(alpha_F) = {1,-1,3,-3}` from witnesses alone.** Witnesses prove inclusion of those classes in the image. The reverse inclusion needs the support lemma showing no squareclass outside the signed squarefree divisors of `b=-3` can occur.

3. **Do not run the dual mod-16 obstruction without primitivity.** The all-even or all-zero residue vector is always a fake solution modulo powers of two. The correct statement uses coprime integral `U,V`, or modulo `16` says `U,V` are not both even.

4. **Do not use the rank formula without the factor `#F(Q)[2]=4`.** The correct relation here is

```text
#im(alpha_F) * #im(alpha_F') = 2^rank * #F(Q)[2] = 2^rank * 4.
```

Equivalently,

```text
2^rank = (#im(alpha_F) * #im(alpha_F')) / 4.
```

5. **Do not treat the dual class `1` as meaning only the identity point.** The dual class `1` contains `(0,0)` by convention and also ordinary affine points such as `(4,8)` on `F'`.

6. **Do not expect the local denominator congruence `B^2 | (C-z^3)(C+z^3)` to replace the descent.** It records prime-by-prime squareclass choices, but the global rank-zero conclusion is exactly what the isogeny/Selmer certificate supplies.

## 7. Recommended implementation boundary

For the current FLT N=12 formalization, the cleanest dependency is:

```text
E_affine_rational_points
```

or, one layer lower,

```text
F_affine_rational_points.
```

The future arithmetic certificate can then prove this theorem using the finite 2-isogeny data above. The existing denominator residual file should only need the elementary wrapper from the rational point list to `B=1`.
