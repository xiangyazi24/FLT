# Q1976 (dm3): N = 16 two-isogeny descent by finite `native_decide` obstructions

## Executive answer

For

```text
E  : w^2 = u^3 - u^2 - u = u*(u^2 - u - 1),
```

there is exactly one nontrivial rational `2`-torsion point, namely `(u,w) = (0,0)`.  The other two roots are `(1 ± sqrt 5)/2`, so they are not rational.

The clean Lean route is therefore not a full split-2-torsion descent.  Use the standard `2`-isogeny descent attached to the rational kernel generated by `(0,0)`.

For a curve

```text
y^2 = x^3 + a*x^2 + b*x,
```

the `2`-isogenous curve is

```text
y^2 = x^3 - 2*a*x^2 + (a^2 - 4*b)*x.
```

For N = 16 this gives

```text
E      : a = -1, b = -1,  y^2 = x^3 - x^2 - x,
E_dual : a =  2, b =  5,  y^2 = x^3 + 2*x^2 + 5*x.
```

The homogeneous covering equation for squareclass `d` is

```text
d*z^2 = d^2*x^4 + a*d*x^2*y^2 + b*y^4.
```

The finite local obstructions are exactly:

```text
E,      d =  5: no primitive solution mod 125;
E,      d = -5: no primitive solution mod 125;
E_dual, d = -1: no primitive solution mod 16;
E_dual, d = -5: no primitive solution mod 16.
```

The remaining classes are the expected trivial/kernel classes:

```text
E      : d =  1, -1;
E_dual : d =  1,  5.
```

Thus the `φ`- and `φ̂`-Selmer groups both have the minimal size forced by the kernel classes, giving rank `0` by the usual `2`-isogeny descent rank formula.  This is indeed simpler than the split/full-2-torsion bookkeeping for N = 14.

## Concrete Lean 4 file

This is a self-contained finite-residue certificate.  It deliberately proves only the local obstruction facts by `native_decide`; the generic wrapper from an integral primitive covering solution to a primitive residue solution modulo `p^k` should be shared with the N = 10 proof.

```lean
import Mathlib

/-!
# Native finite checks for the N = 16 two-isogeny descent

Curve:

  E      : y^2 = x^3 - x^2 - x
  E_dual : y^2 = x^3 + 2*x^2 + 5*x

For `y^2 = x^3 + a*x^2 + b*x`, the homogeneous covering equation for
squareclass `d` is

  d*z^2 = d^2*x^4 + a*d*x^2*y^2 + b*y^4.

The four `native_decide` theorems below are the local obstructions needed for
N = 16:

* on `E`, the classes `d = 5` and `d = -5` are obstructed modulo `125`;
* on `E_dual`, the classes `d = -1` and `d = -5` are obstructed modulo `16`.
-/

namespace MazurProof
namespace N16TwoIsogenyNative

/-- `x,y,z` are primitive at `p`, modulo `m`: at least one is not divisible by `p`. -/
def PrimMod (p : Nat) {m : Nat} (x y z : Fin m) : Prop :=
  x.1 % p ≠ 0 ∨ y.1 % p ≠ 0 ∨ z.1 % p ≠ 0

/--
The covering congruence modulo `m` for

  d*z^2 = d^2*x^4 + a*d*x^2*y^2 + b*y^4.

The variables are represented by residues `Fin m`, then coerced to `ℤ` for the
integer polynomial and reduced modulo `m`.
-/
def CoverEqMod (m : Nat) (a b d : ℤ) (x y z : Fin m) : Prop :=
  (d * (z.1 : ℤ) ^ 2 -
      ((d ^ 2) * (x.1 : ℤ) ^ 4 +
        a * d * (x.1 : ℤ) ^ 2 * (y.1 : ℤ) ^ 2 +
        b * (y.1 : ℤ) ^ 4)) % (m : ℤ) = 0

/-- The N = 16 curve `E : y^2 = x^3 - x^2 - x`, so `a=-1`, `b=-1`. -/
def CoverE16Mod (m : Nat) (d : ℤ) (x y z : Fin m) : Prop :=
  CoverEqMod m (-1) (-1) d x y z

/-- The dual curve `E' : y^2 = x^3 + 2*x^2 + 5*x`, so `a=2`, `b=5`. -/
def CoverE16DualMod (m : Nat) (d : ℤ) (x y z : Fin m) : Prop :=
  CoverEqMod m 2 5 d x y z

/--
On `E`, the squareclass `d = 5` has no primitive local point modulo `5^3`.

Equation checked:

  5*z^2 = 25*x^4 - 5*x^2*y^2 - y^4  mod 125.
-/
theorem E16_d5_no_primitive_mod125 :
    ¬ ∃ x y z : Fin 125,
      PrimMod 5 x y z ∧ CoverE16Mod 125 5 x y z := by
  native_decide

/--
On `E`, the squareclass `d = -5` has no primitive local point modulo `5^3`.

Equation checked:

  -5*z^2 = 25*x^4 + 5*x^2*y^2 - y^4  mod 125.
-/
theorem E16_dm5_no_primitive_mod125 :
    ¬ ∃ x y z : Fin 125,
      PrimMod 5 x y z ∧ CoverE16Mod 125 (-5) x y z := by
  native_decide

/--
On the dual curve `E'`, the squareclass `d = -1` has no primitive local point
modulo `2^4`.

Equation checked:

  -z^2 = x^4 - 2*x^2*y^2 + 5*y^4  mod 16.
-/
theorem E16Dual_dm1_no_primitive_mod16 :
    ¬ ∃ x y z : Fin 16,
      PrimMod 2 x y z ∧ CoverE16DualMod 16 (-1) x y z := by
  native_decide

/--
On the dual curve `E'`, the squareclass `d = -5` has no primitive local point
modulo `2^4`.

Equation checked:

  -5*z^2 = 25*x^4 - 10*x^2*y^2 + 5*y^4  mod 16.
-/
theorem E16Dual_dm5_no_primitive_mod16 :
    ¬ ∃ x y z : Fin 16,
      PrimMod 2 x y z ∧ CoverE16DualMod 16 (-5) x y z := by
  native_decide

end N16TwoIsogenyNative
end MazurProof
```

## How to use this in the descent proof

The generic lemma you want around these checks has this shape:

```text
If an integral primitive covering solution exists for `(a,b,d)`, then for every
modulus `m = p^k` its residue class gives

  ∃ x y z : Fin m, PrimMod p x y z ∧ CoverEqMod m a b d x y z.
```

Once that wrapper exists, the four `native_decide` theorems above immediately remove the nontrivial Selmer candidates:

```text
E      : d =  5, -5 removed by mod 125;
E_dual : d = -1, -5 removed by mod 16.
```

So the only surviving local classes are

```text
E      : {1, -1},
E_dual : {1, 5}.
```

Those are exactly the classes forced by `O` and the rational kernel point `(0,0)` on the corresponding curve.  Hence the `2`-isogeny Selmer dimensions are minimal, and the usual rank formula gives rank `0`.

## Important distinction from the quartic obstruction

The equation

```text
w^2 = u*(u^2 - u - 1)
```

having only one rational `2`-torsion point means the descent is a `2`-isogeny descent, not a full rational split-`2`-torsion descent.  The irrational roots do not need to be represented in Lean.  The finite certificates above only use the rational isogeny model and the homogeneous quartic covering equations, so they avoid adjoining `sqrt 5` entirely.
