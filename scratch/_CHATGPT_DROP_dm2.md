This follows the lightweight `native_decide` pattern of `DescentObstruction.lean`: it verifies the finite local obstruction layer for the N=14 obstruction curve / Cremona curve `96.b3`

```text
E : w^2 = u^3 + u^2 - 2*u,
```

with rational points `O`, `(-2,0)`, `(0,0)`, `(1,0)`.  The global descent bridge from these finite checks to the Mordell-Weil/Selmer conclusion is intentionally left as a later theorem.

import Mathlib.Data.ZMod.Basic

/-!
# Local obstruction checks for the N=14 curve `96.b3`

This file verifies the finite local obstruction checks for the 2-isogeny
descent on

    E  : w² = u³ + u² - 2u,
    E' : W² = U³ - 2U² + 9U.

The original curve has rational Mordell-Weil group `Z/2 × Z/2`, represented by

    O, (-2, 0), (0, 0), (1, 0).

For the 2-isogeny with kernel generated by `(0, 0)`, the descent uses the
standard model

    y² = x³ + a*x² + b*x

and the homogeneous spaces

    d * Y² = d² * X⁴ + d*a*X²*Z² + b*Z⁴.

Thus:

* for `E`,      `a = 1`,  `b = -2`;
* for `E_dual`, `a = -2`, `b = 9`.

The finite checks below are exactly the explicit modular arithmetic layer.  The
formal 2-isogeny descent bridge, the squareclass support theorem, and the final
Selmer/Mordell-Weil conclusion are not supplied here.
-/

namespace FLT
namespace N14LocalObstructions

/-!
## Primitive residue predicates

For a `p`-adic obstruction checked modulo `p^n`, "primitive" means that not
both projective coordinates are divisible by `p`.
-/

def NonzeroMod2_4 (a : ZMod 4) : Bool :=
  decide (a.val % 2 ≠ 0)

def PrimitiveMod2_4 (x z : ZMod 4) : Bool :=
  NonzeroMod2_4 x || NonzeroMod2_4 z

def NonzeroMod2_8 (a : ZMod 8) : Bool :=
  decide (a.val % 2 ≠ 0)

def PrimitiveMod2_8 (x z : ZMod 8) : Bool :=
  NonzeroMod2_8 x || NonzeroMod2_8 z

def NonzeroMod2_16 (a : ZMod 16) : Bool :=
  decide (a.val % 2 ≠ 0)

def PrimitiveMod2_16 (x z : ZMod 16) : Bool :=
  NonzeroMod2_16 x || NonzeroMod2_16 z

def NonzeroMod3_3 (a : ZMod 3) : Bool :=
  decide (a.val % 3 ≠ 0)

def PrimitiveMod3_3 (x z : ZMod 3) : Bool :=
  NonzeroMod3_3 x || NonzeroMod3_3 z

def NonzeroMod3_27 (a : ZMod 27) : Bool :=
  decide (a.val % 3 ≠ 0)

def PrimitiveMod3_27 (x z : ZMod 27) : Bool :=
  NonzeroMod3_27 x || NonzeroMod3_27 z

/-!
## Covering equations

For `E : y² = x³ + x² - 2x`:

    d * Y² = d² * X⁴ + d*X²*Z² - 2*Z⁴.

For the dual `E' : y² = x³ - 2x² + 9x`:

    d * Y² = d² * X⁴ - 2*d*X²*Z² + 9*Z⁴.
-/

def ECover3 (d x z : ZMod 3) : ZMod 3 :=
  d ^ 2 * x ^ 4 +
    d * x ^ 2 * z ^ 2 -
    (2 : ZMod 3) * z ^ 4

def ECover4 (d x z : ZMod 4) : ZMod 4 :=
  d ^ 2 * x ^ 4 +
    d * x ^ 2 * z ^ 2 -
    (2 : ZMod 4) * z ^ 4

def ECover8 (d x z : ZMod 8) : ZMod 8 :=
  d ^ 2 * x ^ 4 +
    d * x ^ 2 * z ^ 2 -
    (2 : ZMod 8) * z ^ 4

def ECover27 (d x z : ZMod 27) : ZMod 27 :=
  d ^ 2 * x ^ 4 +
    d * x ^ 2 * z ^ 2 -
    (2 : ZMod 27) * z ^ 4

def EdualCover8 (d x z : ZMod 8) : ZMod 8 :=
  d ^ 2 * x ^ 4 -
    (2 : ZMod 8) * d * x ^ 2 * z ^ 2 +
    (9 : ZMod 8) * z ^ 4

def EdualCover16 (d x z : ZMod 16) : ZMod 16 :=
  d ^ 2 * x ^ 4 -
    (2 : ZMod 16) * d * x ^ 2 * z ^ 2 +
    (9 : ZMod 16) * z ^ 4

/-!
## No-primitive-solution propositions
-/

def NoPrimitiveE3 (d : ZMod 3) : Prop :=
  ¬ ∃ x y z : ZMod 3,
    PrimitiveMod3_3 x z = true ∧
      d * y ^ 2 = ECover3 d x z

def NoPrimitiveE4 (d : ZMod 4) : Prop :=
  ¬ ∃ x y z : ZMod 4,
    PrimitiveMod2_4 x z = true ∧
      d * y ^ 2 = ECover4 d x z

def NoPrimitiveE8 (d : ZMod 8) : Prop :=
  ¬ ∃ x y z : ZMod 8,
    PrimitiveMod2_8 x z = true ∧
      d * y ^ 2 = ECover8 d x z

def NoPrimitiveE27 (d : ZMod 27) : Prop :=
  ¬ ∃ x y z : ZMod 27,
    PrimitiveMod3_27 x z = true ∧
      d * y ^ 2 = ECover27 d x z

def NoPrimitiveEdual8 (d : ZMod 8) : Prop :=
  ¬ ∃ x y z : ZMod 8,
    PrimitiveMod2_8 x z = true ∧
      d * y ^ 2 = EdualCover8 d x z

def NoPrimitiveEdual16 (d : ZMod 16) : Prop :=
  ¬ ∃ x y z : ZMod 16,
    PrimitiveMod2_16 x z = true ∧
      d * y ^ 2 = EdualCover16 d x z

## `E`-side checks

The rational torsion image on the `E` side is generated by the classes

    1, -2.

Therefore the non-torsion squareclasses supported on the bad primes `{2, 3}`
are

The checks below kill these classes.  Some classes have more than one visible
local obstruction; the package theorem uses one check for each non-surviving
class.

## Dual `E'`-side checks

The rational torsion image on the dual side is

    NoPrimitiveE4 (-1) ∧
    NoPrimitiveE8 2 ∧
    NoPrimitiveE4 3 ∧
