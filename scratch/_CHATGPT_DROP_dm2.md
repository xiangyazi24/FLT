# Q1975 (dm2): concrete `native_decide` checks for the N=14 2-isogeny descent

## Important boundary

You can prove the **finite local obstruction checks** with `native_decide` completely concretely.  But `native_decide` by itself does **not** prove the theorem

```text
2-Selmer group = rational torsion image.
```

That equality still requires the formal 2-isogeny descent bridge saying:

1. every Selmer class is represented by one of the finitely many squareclasses supported on the bad primes;
2. a Selmer class must have points over every completion;
3. the modular no-primitive-solution checks below really rule out the corresponding local completions;
4. the surviving squareclasses are exactly the rational torsion images.

So the concrete Lean code below is the right `native_decide` layer.  It should feed a later bridge theorem, not replace it.

## Squareclass bookkeeping for N=14

For

```text
E  : y^2 = x^3 + x^2 - 2*x = x*(x-1)*(x+2)
E' : y^2 = x^3 - 2*x^2 + 9*x
```

with the 2-isogeny kernel generated by `(0,0)`, use the standard covering convention

```text
d * Y^2 = d^2 * X^4 + d*a*X^2*Z^2 + b*Z^4
```

for a curve

```text
y^2 = x^3 + a*x^2 + b*x.
```

For `E`, we have `a = 1`, `b = -2`, so the covering is

```text
d * Y^2 = d^2 * X^4 + d*X^2*Z^2 - 2*Z^4.
```

The bad-prime squareclasses are represented by

```text
±1, ±2, ±3, ±6.
```

The rational torsion image is generated by the classes

```text
1, -2.
```

Thus the non-torsion `E`-side classes to kill are

```text
-1, 2, 3, -3, 6, -6.
```

For `E'`, we have `a = -2`, `b = 9`, so the covering is

```text
d * Y^2 = d^2 * X^4 - 2*d*X^2*Z^2 + 9*Z^4.
```

The rational torsion image on the dual side is

```text
1, 3
```

because `E'(Q)` contains `O`, `(0,0)`, and `(3,±6)`.  Thus the non-torsion dual classes to kill are

```text
-1, 2, -2, -3, 6, -6.
```

The class `d = -1` on the dual side is killed already at the real place: its equation is

```text
-Y^2 = X^4 + 2X^2Z^2 + 9Z^4,
```

which has no nonzero real solution.  The remaining dual classes are killed by `native_decide` modulo `8` or `16`.

## Concrete Lean 4 code

```lean
import Mathlib.Data.ZMod.Basic
import Mathlib.Tactic

namespace FLT
namespace N14LocalSelmer

/-!
Concrete local obstruction checks for the N=14 obstruction curve

  E  : y² = x³ + x² - 2x,
  E' : y² = x³ - 2x² + 9x.

These checks prove the finite local failures of the non-torsion 2-isogeny
covering classes.  A separate 2-isogeny-descent bridge is still needed to turn
these into the statement that the corresponding Selmer groups equal the rational
torsion images.
-/

/-! ## Primitive residue predicates -/

def NonzeroMod2_4 (a : ZMod 4) : Bool :=
  decide (a.val % 2 ≠ 0)

def PrimitiveMod2_4 (x z : ZMod 4) : Bool :=
  NonzeroMod2_4 x || NonzeroMod2_4 z

def NonzeroMod2_8 (a : ZMod 8) : Bool :=
  decide (a.val % 2 ≠ 0)

def PrimitiveMod2_8 (x z : ZMod 8) : Bool :=
  NonzeroMod2_8 x || NonzeroMod2_8 z

def NonzeroMod2_16 (a : ZMod 16) : Bool :=
  decide (a.val % 2 ≠ 0)

def PrimitiveMod2_16 (x z : ZMod 16) : Bool :=
  NonzeroMod2_16 x || NonzeroMod2_16 z

def NonzeroMod3_27 (a : ZMod 27) : Bool :=
  decide (a.val % 3 ≠ 0)

def PrimitiveMod3_27 (x z : ZMod 27) : Bool :=
  NonzeroMod3_27 x || NonzeroMod3_27 z

/-! ## E-side covers

For `E : y² = x³ + x² - 2x`, the covering convention is

  d*Y² = d²*X⁴ + d*X²*Z² - 2*Z⁴.

The torsion image is `{1, -2}`.  The following checks kill the other classes
`-1, 2, 3, -3, 6, -6`.
-/

/-- E-side class `d = -1`, killed modulo `4`. -/
set_option maxHeartbeats 0 in
theorem E_dneg1_no_primitive_mod4 :
    ¬ ∃ x y z : ZMod 4,
      PrimitiveMod2_4 x z = true ∧
        (-1 : ZMod 4) * y ^ 2 =
          x ^ 4 - x ^ 2 * z ^ 2 -
          (2 : ZMod 4) * z ^ 4 := by
  native_decide

/-- E-side class `d = 2`, killed modulo `8`. -/
set_option maxHeartbeats 0 in
theorem E_d2_no_primitive_mod8 :
    ¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (2 : ZMod 8) * y ^ 2 =
          (4 : ZMod 8) * x ^ 4 +
          (2 : ZMod 8) * x ^ 2 * z ^ 2 -
          (2 : ZMod 8) * z ^ 4 := by
  native_decide

/-- E-side class `d = 3`, killed modulo `4`. -/
set_option maxHeartbeats 0 in
theorem E_d3_no_primitive_mod4 :
    ¬ ∃ x y z : ZMod 4,
      PrimitiveMod2_4 x z = true ∧
        (3 : ZMod 4) * y ^ 2 =
          (9 : ZMod 4) * x ^ 4 +
          (3 : ZMod 4) * x ^ 2 * z ^ 2 -
          (2 : ZMod 4) * z ^ 4 := by
  native_decide

/-- E-side class `d = -3`, killed modulo `27`. -/
set_option maxHeartbeats 0 in
theorem E_dneg3_no_primitive_mod27 :
    ¬ ∃ x y z : ZMod 27,
      PrimitiveMod3_27 x z = true ∧
        (-3 : ZMod 27) * y ^ 2 =
          (9 : ZMod 27) * x ^ 4 -
          (3 : ZMod 27) * x ^ 2 * z ^ 2 -
          (2 : ZMod 27) * z ^ 4 := by
  native_decide

/-- E-side class `d = 6`, killed modulo `27`. -/
set_option maxHeartbeats 0 in
theorem E_d6_no_primitive_mod27 :
    ¬ ∃ x y z : ZMod 27,
      PrimitiveMod3_27 x z = true ∧
        (6 : ZMod 27) * y ^ 2 =
          (36 : ZMod 27) * x ^ 4 +
          (6 : ZMod 27) * x ^ 2 * z ^ 2 -

  native_decide

/-- E-side class `d = -6`, killed modulo `8`. -/
set_option maxHeartbeats 0 in

    ¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (-6 : ZMod 8) * y ^ 2 =
          (36 : ZMod 8) * x ^ 4 -
          (6 : ZMod 8) * x ^ 2 * z ^ 2 -
          (2 : ZMod 8) * z ^ 4 := by
  native_decide

/-! ## Dual E'-side covers

For `E' : y² = x³ - 2x² + 9x`, the covering convention is

  d*Y² = d²*X⁴ - 2*d*X²*Z² + 9*Z⁴.

The rational torsion image is `{1, 3}`.  The real place kills `d = -1`; the
following finite checks kill `2, -2, -3, 6, -6`.
-/

/-- Dual class `d = 2`, killed modulo `8`. -/
set_option maxHeartbeats 0 in
theorem Ep_d2_no_primitive_mod8 :
    ¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (2 : ZMod 8) * y ^ 2 =
          (4 : ZMod 8) * x ^ 4 -
          (4 : ZMod 8) * x ^ 2 * z ^ 2 +
          (9 : ZMod 8) * z ^ 4 := by
  native_decide

/-- Dual class `d = -2`, killed modulo `8`. -/
set_option maxHeartbeats 0 in
theorem Ep_dneg2_no_primitive_mod8 :
    ¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (-2 : ZMod 8) * y ^ 2 =
          (4 : ZMod 8) * x ^ 4 +
          (4 : ZMod 8) * x ^ 2 * z ^ 2 +
          (9 : ZMod 8) * z ^ 4 := by
  native_decide

/-- Dual class `d = -3`, killed modulo `16`. -/
set_option maxHeartbeats 0 in
theorem Ep_dneg3_no_primitive_mod16 :
    ¬ ∃ x y z : ZMod 16,
      PrimitiveMod2_16 x z = true ∧
        (-3 : ZMod 16) * y ^ 2 =
          (9 : ZMod 16) * x ^ 4 +
          (6 : ZMod 16) * x ^ 2 * z ^ 2 +
          (9 : ZMod 16) * z ^ 4 := by
  native_decide

/-- Dual class `d = 6`, killed modulo `8`. -/
set_option maxHeartbeats 0 in

    ¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (6 : ZMod 8) * y ^ 2 =
          (36 : ZMod 8) * x ^ 4 -
          (12 : ZMod 8) * x ^ 2 * z ^ 2 +
          (9 : ZMod 8) * z ^ 4 := by
  native_decide

/-- Dual class `d = -6`, killed modulo `8`. -/
set_option maxHeartbeats 0 in
theorem Ep_dneg6_no_primitive_mod8 :
    ¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (-6 : ZMod 8) * y ^ 2 =
          (36 : ZMod 8) * x ^ 4 +
          (12 : ZMod 8) * x ^ 2 * z ^ 2 +

/-! ## Package the finite checks -/

/-- All finite local obstruction checks for the non-torsion E-side classes. -/
theorem E_side_non_torsion_classes_locally_obstructed :
    (¬ ∃ x y z : ZMod 4,
      PrimitiveMod2_4 x z = true ∧
        (-1 : ZMod 4) * y ^ 2 =
          x ^ 4 - x ^ 2 * z ^ 2 -
          (2 : ZMod 4) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (2 : ZMod 8) * y ^ 2 =
          (4 : ZMod 8) * x ^ 4 +
          (2 : ZMod 8) * x ^ 2 * z ^ 2 -
          (2 : ZMod 8) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 4,
      PrimitiveMod2_4 x z = true ∧
        (3 : ZMod 4) * y ^ 2 =
          (9 : ZMod 4) * x ^ 4 +
          (3 : ZMod 4) * x ^ 2 * z ^ 2 -
          (2 : ZMod 4) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 27,
      PrimitiveMod3_27 x z = true ∧
        (-3 : ZMod 27) * y ^ 2 =
          (9 : ZMod 27) * x ^ 4 -
          (3 : ZMod 27) * x ^ 2 * z ^ 2 -
          (2 : ZMod 27) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 27,
      PrimitiveMod3_27 x z = true ∧
        (6 : ZMod 27) * y ^ 2 =
          (36 : ZMod 27) * x ^ 4 +
          (6 : ZMod 27) * x ^ 2 * z ^ 2 -
          (2 : ZMod 27) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (-6 : ZMod 8) * y ^ 2 =
          (36 : ZMod 8) * x ^ 4 -
          (6 : ZMod 8) * x ^ 2 * z ^ 2 -
          (2 : ZMod 8) * z ^ 4) := by
  constructor
  · exact E_dneg1_no_primitive_mod4
  constructor
  · exact E_d2_no_primitive_mod8
  constructor
  · exact E_d3_no_primitive_mod4
  constructor
  · exact E_dneg3_no_primitive_mod27
  constructor
  · exact E_d6_no_primitive_mod27
  · exact E_dneg6_no_primitive_mod8

/-- All finite local obstruction checks for the non-torsion dual E'-side classes except `d=-1`.
The class `d=-1` is excluded by the real place, not by one of these finite checks. -/
theorem Ep_side_non_torsion_classes_locally_obstructed_except_real_dneg1 :
    (¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (2 : ZMod 8) * y ^ 2 =
          (4 : ZMod 8) * x ^ 4 -
          (4 : ZMod 8) * x ^ 2 * z ^ 2 +
          (9 : ZMod 8) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (-2 : ZMod 8) * y ^ 2 =
          (4 : ZMod 8) * x ^ 4 +
          (4 : ZMod 8) * x ^ 2 * z ^ 2 +
          (9 : ZMod 8) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 16,
      PrimitiveMod2_16 x z = true ∧
        (-3 : ZMod 16) * y ^ 2 =
          (9 : ZMod 16) * x ^ 4 +
          (6 : ZMod 16) * x ^ 2 * z ^ 2 +
          (9 : ZMod 16) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (6 : ZMod 8) * y ^ 2 =
          (36 : ZMod 8) * x ^ 4 -
          (12 : ZMod 8) * x ^ 2 * z ^ 2 +
          (9 : ZMod 8) * z ^ 4)
    ∧
    (¬ ∃ x y z : ZMod 8,
      PrimitiveMod2_8 x z = true ∧
        (-6 : ZMod 8) * y ^ 2 =
          (36 : ZMod 8) * x ^ 4 +
          (12 : ZMod 8) * x ^ 2 * z ^ 2 +
          (9 : ZMod 8) * z ^ 4) := by
  constructor
  · exact Ep_d2_no_primitive_mod8
  constructor
  · exact Ep_dneg2_no_primitive_mod8
  constructor
  · exact Ep_dneg3_no_primitive_mod16
  constructor
  · exact Ep_d6_no_primitive_mod8
  · exact Ep_dneg6_no_primitive_mod8

end N14LocalSelmer
end FLT
```

## How to connect this to “Selmer equals torsion”

The key correction relative to a minimal p=2/p=3 answer is that, to prove the **whole** Selmer equality, you should kill all non-torsion squareclasses supported on `{2,3}`:

```text
E-side survivors:       1, -2
E-side killed:          -1, 2, 3, -3, 6, -6

E'-side survivors:      1, 3
E'-side killed:         -1 by R, and 2, -2, -3, 6, -6 by finite native_decide checks
```

This gives the concrete modular arithmetic layer needed for the N=14 rank-zero/torsion proof.  The remaining formal work is the descent bridge from these local failures to the Selmer equality statement.
