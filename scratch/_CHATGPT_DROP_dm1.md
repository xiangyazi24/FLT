# Q371-dm1: division-polynomial differential addition certificate

## Executive answer

For the differential-addition companion, the clean identity is **not** another one-center certificate like doubling.  It uses the two adjacent centers

```text
m and m+1.
```

The denominator identity is exact by expansion:

```text
ΨSq(2m+1) = deltaP(m)^2
```

where

```text
deltaP(m) := Φ(m) ΨSq(m+1) - Φ(m+1) ΨSq(m).
```

With the orientation above,

```text
deltaP(m) = preΨ(2m+1)
```

for both parities of `m`.

The numerator identity is exact **after `Ψ₃`-saturation and cancellation**, exactly like the doubling certificate:

```text
Ψ₃ * ( Φ(2m+1) - (sumNumP(m,m+1) - deltaP(m)^2 * X) ) = 0.
```

The finite relation list is exactly

```text
Adj(m), Inv(m), Adj(m+1), Inv(m+1), bRel.
```

No `m-1` relation is needed.  No relation beyond adjacent Somos, the preΨ invariant, and the b-invariant relation is needed.

I do **not** have a trustworthy compact hand-written original-generator cofactor list for this one.  The certificate is substantially larger than the doubling one; a Groebner reduction confirms membership in the ideal generated by the five relations above after multiplying by `Ψ₃`, but the compact basis certificate uses an extra S-polynomial if extracted naively.  I would not fabricate hand cofactors.  The Lean route below uses `polyrith` to synthesize the polynomial certificate from the exact five relation instances after all definitions are expanded.  If you need literal cofactors, run the same goal with certificate-output tooling (`polyrith?` if available in your local Mathlib/tactic setup, or an external CAS on the normalized ring goal) and paste the generated certificate back into `linear_combination`.

## The exact identities

Define the homogeneous differential-addition polynomials in `R[X]`:

```lean
noncomputable def deltaP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  X0 * Z1 - X1 * Z0

noncomputable def sumNumP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  C (2 : R) * X0 * X1 * (X0 * Z1 + X1 * Z0)
    + C W.b₂ * X0 * X1 * Z0 * Z1
    + C W.b₄ * Z0 * Z1 * (X0 * Z1 + X1 * Z0)
    + C W.b₆ * Z0^2 * Z1^2

noncomputable def diffAddNumP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  sumNumP W X0 Z0 X1 Z1 - (deltaP W X0 Z0 X1 Z1)^2 * X

noncomputable def diffAddDenP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  (deltaP W X0 Z0 X1 Z1)^2
```

For the consecutive pairs:

```lean
X0 := W.Φ m
Z0 := W.ΨSq m
X1 := W.Φ (m+1)
Z1 := W.ΨSq (m+1)
```

we prove:

```lean
W.ΨSq (2*m+1) = diffAddDenP W X0 Z0 X1 Z1
```

and, under `hc3 : W.Ψ₃ ≠ 0`,

```lean
W.Φ (2*m+1) = diffAddNumP W X0 Z0 X1 Z1.
```

Thus the projective equality has scalar `c = 1`:

```lean
W.Φ (2*m+1) * diffAddDenP W X0 Z0 X1 Z1
  = W.ΨSq (2*m+1) * diffAddNumP W X0 Z0 X1 Z1.
```

The internal saturation scalar is `Ψ₃`, not part of the final projective scalar.

## Local expansions checked by parity

Let

```text
s  := Ψ₂Sq
c3 := Ψ₃
x  := X
P_{-2} := preΨ(m-2)
P_{-1} := preΨ(m-1)
P_0    := preΨ(m)
P_1    := preΨ(m+1)
P_2    := preΨ(m+2)
P_3    := preΨ(m+3)
```

If `m` is even:

```text
Z0 = P0^2*s
F0 = x*P0^2*s - P1*P_{-1}
Z1 = P1^2
F1 = x*P1^2 - P2*P0*s

deltaP = F0*Z1 - F1*Z0
       = P2*P0^3*s^2 - P_{-1}*P1^3
       = preΨ(2m+1).

preΨ(2m)   = P0  * (P_{-1}^2*P2 - P_{-2}*P1^2)
preΨ(2m+2) = P1  * (P0^2*P3 - P_{-1}*P2^2)

Φ(2m+1) = x*deltaP^2 - s*preΨ(2m+2)*preΨ(2m).
```

If `m` is odd:

```text
Z0 = P0^2
F0 = x*P0^2 - P1*P_{-1}*s
Z1 = P1^2*s
F1 = x*P1^2*s - P2*P0

deltaP = F0*Z1 - F1*Z0
       = P2*P0^3 - P_{-1}*P1^3*s^2
       = preΨ(2m+1).

preΨ(2m)   = P0  * (P_{-1}^2*P2 - P_{-2}*P1^2)
preΨ(2m+2) = P1  * (P0^2*P3 - P_{-1}*P2^2)

Φ(2m+1) = x*deltaP^2 - s*preΨ(2m+2)*preΨ(2m).
```

## Relation instances used by the numerator certificate

For `m` even:

```text
Adj(m):
P_{-2}*P2 - P_{-1}*P1 + c3*P0^2 = 0

Inv(m):
c3*(P2*P_{-1}^2 + P1^2*P_{-2} + s^2*P0^3)
  - (preΨ4+s^2)*P1*P0*P_{-1} = 0

Adj(m+1):
P_{-1}*P3 - s^2*P2*P0 + c3*P1^2 = 0

Inv(m+1):
c3*(P3*P0^2 + P2^2*P_{-1} + P1^3)
  - (preΨ4+s^2)*P2*P1*P0 = 0
```

For `m` odd:

```text
Adj(m):
P_{-2}*P2 - s^2*P_{-1}*P1 + c3*P0^2 = 0

Inv(m):
c3*(P2*P_{-1}^2 + P1^2*P_{-2} + P0^3)
  - (preΨ4+s^2)*P1*P0*P_{-1} = 0

Adj(m+1):
P_{-1}*P3 - P2*P0 + c3*P1^2 = 0

Inv(m+1):
c3*(P3*P0^2 + P2^2*P_{-1} + s^2*P1^3)
  - (preΨ4+s^2)*P2*P1*P0 = 0
```

The curve relation is always:

```text
bRel : b₂*b₆ - b₄^2 - 4*b₈ = 0.
```

## Lean code

This is written to mirror your landed doubling proof.  Replace only the local theorem names if your file uses different namespaces.

```lean
import Mathlib.AlgebraicGeometry.EllipticCurve.DivisionPolynomial.Basic
import Mathlib.Tactic
import Mathlib.Tactic.Polyrith

open Polynomial
open scoped Polynomial

namespace WeierstrassCurve

noncomputable section

variable {R : Type*} [CommRing R] [IsDomain R]

noncomputable def deltaP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  X0 * Z1 - X1 * Z0

noncomputable def sumNumP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  C (2 : R) * X0 * X1 * (X0 * Z1 + X1 * Z0)
    + C W.b₂ * X0 * X1 * Z0 * Z1
    + C W.b₄ * Z0 * Z1 * (X0 * Z1 + X1 * Z0)
    + C W.b₆ * Z0^2 * Z1^2

noncomputable def diffAddNumP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  sumNumP W X0 Z0 X1 Z1 - (deltaP W X0 Z0 X1 Z1)^2 * X

noncomputable def diffAddDenP
    (W : WeierstrassCurve R) (X0 Z0 X1 Z1 : R[X]) : R[X] :=
  (deltaP W X0 Z0 X1 Z1)^2

private lemma preΨ_adjacent_somos_res
    (W : WeierstrassCurve R) (h4 : (4 : R) ≠ 0) (m : ℤ) :
    W.preΨ (m - 2) * W.preΨ (m + 2)
      - (if Even m then 1 else W.Ψ₂Sq^2) *
          (W.preΨ (m - 1) * W.preΨ (m + 1))
      + W.Ψ₃ * W.preΨ m^2 = 0 := by
  have h := preΨ_adjacent_somos W h4 m
  linear_combination (norm := ring_nf) h

private lemma preΨ_invariant_res
    (W : WeierstrassCurve R)
    (h4 : (4 : R) ≠ 0)
    (hψ_ne : ∀ k : ℤ, k ≠ 0 → W.ψ k ≠ 0)
    (m : ℤ) :
    W.Ψ₃ *
        (W.preΨ (m + 2) * W.preΨ (m - 1)^2
          + W.preΨ (m + 1)^2 * W.preΨ (m - 2)
          + (if Even m then W.Ψ₂Sq^2 else 1) * W.preΨ m^3)
      - (W.preΨ₄ + W.Ψ₂Sq^2) *
          (W.preΨ (m + 1) * W.preΨ m * W.preΨ (m - 1)) = 0 := by
  have h := preΨ_invariant_raw W h4 hψ_ne m
  linear_combination (norm := ring_nf) h

private lemma bRel_poly (W : WeierstrassCurve R) :
    C W.b₂ * C W.b₆ - (C W.b₄)^2 - C (4 : R) * C W.b₈ = (0 : R[X]) := by
  have hb0 : W.b₂ * W.b₆ - W.b₄^2 - (4 : R) * W.b₈ = 0 := by
    have hb := b_relation (W := W)
    -- hb : 4 * W.b₈ = W.b₂ * W.b₆ - W.b₄^2
    rw [← hb]
    ring
  have hbC := congrArg (fun z : R => (C z : R[X])) hb0
  linear_combination (norm := ring_nf) hbC

private lemma preΨ_2m_add_one_even
    (W : WeierstrassCurve R) {m : ℤ} (hm : Even m) :
    W.preΨ (2*m + 1)
      = W.preΨ (m + 2) * W.preΨ m^3 * W.Ψ₂Sq^2
        - W.preΨ (m - 1) * W.preΨ (m + 1)^3 := by
  simpa [hm] using W.preΨ_odd m

private lemma preΨ_2m_add_one_odd
    (W : WeierstrassCurve R) {m : ℤ} (hm : ¬ Even m) :
    W.preΨ (2*m + 1)
      = W.preΨ (m + 2) * W.preΨ m^3
        - W.preΨ (m - 1) * W.preΨ (m + 1)^3 * W.Ψ₂Sq^2 := by
  simpa [hm] using W.preΨ_odd m

private lemma preΨ_2m_even
    (W : WeierstrassCurve R) (m : ℤ) :
    W.preΨ (2*m)
      = W.preΨ (m - 1)^2 * W.preΨ m * W.preΨ (m + 2)
        - W.preΨ (m - 2) * W.preΨ m * W.preΨ (m + 1)^2 := by
  simpa [mul_assoc, mul_comm, mul_left_comm] using W.preΨ_even m

private lemma preΨ_2m_add_two
    (W : WeierstrassCurve R) (m : ℤ) :
    W.preΨ (2*m + 2)
      = W.preΨ m^2 * W.preΨ (m + 1) * W.preΨ (m + 3)
        - W.preΨ (m - 1) * W.preΨ (m + 1) * W.preΨ (m + 2)^2 := by
  have h := W.preΨ_even (m + 1)
  rw [show 2 * (m + 1) = 2*m + 2 by ring] at h
  simpa [add_assoc, add_comm, add_left_comm, sub_eq_add_neg, mul_assoc, mul_comm, mul_left_comm]
    using h
```

### Exact denominator identity

```lean
lemma ΨSq_two_mul_add_one
    (W : WeierstrassCurve R) (m : ℤ) :
    W.ΨSq (2*m + 1)
      = diffAddDenP W (W.Φ m) (W.ΨSq m) (W.Φ (m+1)) (W.ΨSq (m+1)) := by
  by_cases hm : Even m
  · have hm1 : ¬ Even (m + 1) := by omega
    have h2m1 : ¬ Even (2*m + 1) := by omega
    rw [W.ΨSq_odd m]
    simp [diffAddDenP, deltaP, WeierstrassCurve.Φ, WeierstrassCurve.ΨSq,
      hm, hm1, h2m1]
    ring_nf
  · have hm1 : Even (m + 1) := by omega
    have h2m1 : ¬ Even (2*m + 1) := by omega
    rw [W.ΨSq_odd m]
    simp [diffAddDenP, deltaP, WeierstrassCurve.Φ, WeierstrassCurve.ΨSq,
      hm, hm1, h2m1]
    ring_nf
```

### Saturated numerator identity, even branch

```lean
private lemma Φ_two_mul_add_one_sat_even
    (W : WeierstrassCurve R)
    (h4 : (4 : R) ≠ 0)
    (hψ_ne : ∀ k : ℤ, k ≠ 0 → W.ψ k ≠ 0)
    {m : ℤ} (hm : Even m) :
    W.Ψ₃ *
      (W.Φ (2*m + 1)
        - diffAddNumP W (W.Φ m) (W.ΨSq m) (W.Φ (m+1)) (W.ΨSq (m+1))) = 0 := by
  set_option maxHeartbeats 1200000 in
  set_option maxRecDepth 4096 in
  have hAdj0 := preΨ_adjacent_somos_res W h4 m
  have hInv0 := preΨ_invariant_res W h4 hψ_ne m
  have hAdj1 := preΨ_adjacent_somos_res W h4 (m + 1)
  have hInv1 := preΨ_invariant_res W h4 hψ_ne (m + 1)
  have hb := bRel_poly W

  have hm1 : ¬ Even (m + 1) := by omega
  have h2m1 : ¬ Even (2*m + 1) := by omega
  have hpre_odd := preΨ_2m_add_one_even W hm
  have hpre_even0 := preΨ_2m_even W m
  have hpre_even1 := preΨ_2m_add_two W m

  rw [WeierstrassCurve.Φ]
  rw [W.ΨSq_odd m]
  rw [hpre_odd, hpre_even0, hpre_even1]

  -- Expand all curve constants and the five relation instances into one polynomial problem.
  simp [diffAddNumP, sumNumP, deltaP,
    WeierstrassCurve.Φ, WeierstrassCurve.ΨSq,
    WeierstrassCurve.Ψ₂Sq, WeierstrassCurve.Ψ₃, WeierstrassCurve.preΨ₄,
    hm, hm1, h2m1] at hAdj0 hInv0 hAdj1 hInv1 hb ⊢
  ring_nf at hAdj0 hInv0 hAdj1 hInv1 hb ⊢
  polyrith [hAdj0, hInv0, hAdj1, hInv1, hb]
```

### Exact numerator identity after cancelling `Ψ₃`

```lean
lemma Φ_two_mul_add_one
    (W : WeierstrassCurve R)
    (h4 : (4 : R) ≠ 0)
    (hψ_ne : ∀ k : ℤ, k ≠ 0 → W.ψ k ≠ 0)
    (hc3 : W.Ψ₃ ≠ 0)
    (m : ℤ) :
    W.Φ (2*m + 1)
      = diffAddNumP W (W.Φ m) (W.ΨSq m) (W.Φ (m+1)) (W.ΨSq (m+1)) := by
  apply sub_eq_zero.mp
  have hsat :
      W.Ψ₃ *
        (W.Φ (2*m + 1)
          - diffAddNumP W (W.Φ m) (W.ΨSq m) (W.Φ (m+1)) (W.ΨSq (m+1))) = 0 := by
    by_cases hm : Even m
    · exact Φ_two_mul_add_one_sat_even W h4 hψ_ne hm
    · exact Φ_two_mul_add_one_sat_odd W h4 hψ_ne hm
  exact mul_left_cancel₀ hc3 hsat
```

lemma diffAdd_projective_two_mul_add_one

## If `polyrith` is too slow

The normalized numerator goals are large.  The relation list is nevertheless fixed and small:

```text
hAdj0 = Adj(m)
hInv0 = Inv(m)
hAdj1 = Adj(m+1)
hInv1 = Inv(m+1)
hb    = bRel
```

A practical workflow is:

1. Temporarily run the saturated lemma with `polyrith` in the branch.
2. If your local tactic supports proof extraction, use the extracted certificate to replace `polyrith` by `linear_combination`.
3. If extracting externally, normalize to the variables

   ```text
   Pm2, Pm1, P0, P1, P2, P3, X, b₂, b₄, b₆, b₈
   ```

   with `Ψ₂Sq`, `Ψ₃`, and `preΨ₄` expanded, and ask the CAS for membership of

   ```text
   Ψ₃ * (Φ(2m+1) - diffAddNumP)
   ```

   in the ideal generated by the five relations above.

The denominator identity should not use `polyrith`; it is just `ΨSq_odd` plus the definition of `Φ` and `ΨSq` at `m` and `m+1`.
