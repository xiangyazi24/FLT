# Q301-dm1: Ward theorem formalization — gap/off mutual induction certificates

This note gives **polynomial-residual orientations and exact cofactors**.  I use residuals rather than propositions so the signs are unambiguous for `linear_combination`.

Reference anchors:

```text
van der Poorten--Swart, "Recurrence Relations for Elliptic Sequences: Every Somos 4 is a Somos k",
Section 6, Theorem 3, equations (13), (14), and the displayed identities (15) and the following product identity.

Stange, "Elliptic nets and elliptic curves", Definition 1.1 / recurrence (4).
```

Stange's recurrence is

```text
EN(p,q,r,s):
W(p+q+s)W(p-q)W(r+s)W(r)
+ W(q+r+s)W(q-r)W(p+s)W(p)
+ W(r+p+s)W(r-p)W(q+s)W(q) = 0.
```

The off-diagonal relation below is exactly `EN(m,k,1,1)`.

---

## 0. Notation and residual orientations

Throughout:

```text
b := W(2)
B(i) := W(i+1) W(i-1)
U(k,m) := W(m+k) W(m-k)
V(k,m) := b W(m+k+1) W(m-k)
```

Closed right-hand sides:

```text
GR(k,m) := B(m) W(k)^2 - B(k) W(m)^2

HR(k,m) := W(m+2) W(m-1) W(k+1) W(k)
         - W(k+2) W(k-1) W(m+1) W(m)
```

Residuals:

```text
GapRes(k,m) := U(k,m) - GR(k,m)
OffRes(k,m) := V(k,m) - HR(k,m)
```

Adjacent Somos residual:

```text
S(i) := W(i+2) W(i-2) - b^2 W(i+1) W(i-1) + c W(i)^2
```

Invariant/Trel notation:

```text
N(i) := W(i+2) W(i-1)^2 + W(i+1)^2 W(i-2) + b^2 W(i)^3
D(i) := W(i+1) W(i) W(i-1)
T(i,j) := N(i) D(j) - N(j) D(i)
Inv(i) := c N(i) - (d + b^4) D(i)
```

All certificates below are identities in the polynomial ring generated by the displayed local values plus the combined-index variables appearing in `U` and `V`.

---

## 1. Exact off-diagonal companion

The companion invariant is:

```text
OffRel(k,m):
V(k,m) = HR(k,m)
```

Expanded:

```text
b W(m+k+1) W(m-k)
=
W(m+2) W(m-1) W(k+1) W(k)
- W(k+2) W(k-1) W(m+1) W(m).
```

Why this is the EDS/net identity: set

```text
p = m, q = k, r = 1, s = 1
```

in Stange's recurrence.  Since `W(1)=1` and `W(1-m)=-W(m-1)`, the instance is

```text
0 = W(m+k+1)W(m-k)W(2)W(1)
  + W(k+2)W(k-1)W(m+1)W(m)
  + W(m+2)W(1-m)W(k+1)W(k),
```

hence exactly

```text
b W(m+k+1) W(m-k)
=
W(m+2) W(m-1) W(k+1) W(k)
- W(k+2) W(k-1) W(m+1) W(m).
```

In van der Poorten--Swart notation this is Theorem 3, equation (14), after setting `A = W`, `W_1 = 1`, and replacing their gap variable by `k`.

---

## 2. Mutual induction order

Use induction on nonnegative gap `k`, proving the pair

```text
G_k : ∀ m, GapRes(k,m) = 0
H_k : ∀ m, OffRes(k,m) = 0
```

with the following stage order:

```text
Given G_k and H_k for all centers m:
  first prove G_{k+1} for all centers m using H_k(m), H_k(m-1), G_k(m), S(m), S(k), T(k,m);
  then prove H_{k+1} for all centers m using H_k(m), G_{k+1}(m), G_{k+1}(m+1), S(m), S(k+1), T(k+1,m).
```

Important: the H-step genuinely needs the shifted center `GapRel(k+1,m+1)`.  Since the G-step has just produced `G_{k+1}` for **all** centers, this is already available.

---

## 3. G-step certificate: derive `GapRel(k+1,m)`

Tautology orientation:

```text
TauG(k,m) := b^2 U(k,m) U(k+1,m) - V(k,m) V(k,m-1)
```

The target residual is

```text
GapRes(k+1,m) = U(k+1,m) - GR(k+1,m).
```

The exact multiplied identity is:

```text
b^2 U(k,m) * GapRes(k+1,m)
=
  TauG(k,m)
+ V(k,m-1) * OffRes(k,m)
+ HR(k,m) * OffRes(k,m-1)
- b^2 GR(k+1,m) * GapRes(k,m)
+ W(k)^2 * W(k+1)^2 * W(m+1) * W(m-1) * S(m)
+ W(m) * W(k+2) * T(k,m)
- W(k+1)^2 * W(m+1) * W(m)^2 * W(m-1) * S(k).
```

So the linear-combination cofactors are exactly:

```text
hypothesis/residual              cofactor
---------------------------------------------------------------
TauG(k,m)                        1
OffRes(k,m)                      V(k,m-1)
OffRes(k,m-1)                    HR(k,m)
GapRes(k,m)                      - b^2 GR(k+1,m)
S(m)                             W(k)^2 W(k+1)^2 W(m+1) W(m-1)
T(k,m)                           W(m) W(k+2)
S(k)                             - W(k+1)^2 W(m+1) W(m)^2 W(m-1)
```

The residual-only part is the eight-term obstruction you saw, with the following exact closure:

```text
HR(k,m) HR(k,m-1) - b^2 GR(k,m) GR(k+1,m)
=
  W(k)^2 W(k+1)^2 W(m+1) W(m-1) S(m)
+ W(m) W(k+2) T(k,m)
- W(k+1)^2 W(m+1) W(m)^2 W(m-1) S(k).
```

In the local-symbol convention

```text
K_i := W(k+i),   M_i := W(m+i),
```

that closure is

```text
HR(k,m) HR(k,m-1) - b^2 GR(k,m) GR(k+1,m)
=
K_0^2 K_1^2 M_1 M_{-1} S(m)
+ M_0 K_2 T(k,m)
- K_1^2 M_1 M_0^2 M_{-1} S(k).
```

where

```text
S(m) = M_2 M_{-2} - b^2 M_1 M_{-1} + c M_0^2,
S(k) = K_2 K_{-2} - b^2 K_1 K_{-1} + c K_0^2,
T(k,m) = N(k)D(m)-N(m)D(k).
```

Cancellation factor for the generic G-step:

```text
b^2 U(k,m) = b^2 W(m+k) W(m-k).
```

This factor is not nonzero on the diagonal/anti-diagonal `m = ±k`.  In a universal-domain proof, split those cases.  For `m = k`, the target `GapRel(k+1,k)` is exactly the odd normEDS recurrence

```text
W(2k+1) = W(k+2) W(k)^3 - W(k+1)^3 W(k-1),
```

up to the sign `W(-1)=-1`; the case `m = -k` is the same by antisymmetry.

## 4. H-step certificate: derive `OffRel(k+1,m)`

Use the product tautology

```text
V(k,m) V(k+1,m) = b^2 U(k+1,m+1) U(k+1,m).
```

Orient it as

```text
TauH(k,m) := V(k,m) V(k+1,m) - b^2 U(k+1,m+1) U(k+1,m).
```

The target residual is

```text
OffRes(k+1,m) = V(k+1,m) - HR(k+1,m).
```

The exact multiplied identity is:

```text
V(k,m) * OffRes(k+1,m)
=
  TauH(k,m)
- HR(k+1,m) * OffRes(k,m)
+ b^2 U(k+1,m+1) * GapRes(k+1,m)
+ b^2 GR(k+1,m) * GapRes(k+1,m+1)
+ W(k) * W(k+2) * W(k+1)^2 * W(m+1)^2 * S(m)
+ W(k+1) * W(m+2) * T(k+1,m)
- W(k) * W(k+2) * W(m)^2 * W(m+1)^2 * S(k+1).
```

So the linear-combination cofactors are exactly:

```text
hypothesis/residual              cofactor
---------------------------------------------------------------------
TauH(k,m)                        1
OffRes(k,m)                      - HR(k+1,m)
GapRes(k+1,m)                    b^2 U(k+1,m+1)
GapRes(k+1,m+1)                  b^2 GR(k+1,m)
S(m)                             W(k) W(k+2) W(k+1)^2 W(m+1)^2
T(k+1,m)                         W(k+1) W(m+2)
S(k+1)                           - W(k) W(k+2) W(m)^2 W(m+1)^2
```

The residual-only closure is:

```text
b^2 GR(k+1,m+1) GR(k+1,m) - HR(k+1,m) HR(k,m)
=
  W(k) W(k+2) W(k+1)^2 W(m+1)^2 S(m)
+ W(k+1) W(m+2) T(k+1,m)
- W(k) W(k+2) W(m)^2 W(m+1)^2 S(k+1).
```

In the local-symbol convention

```text
K_i := W(k+i),   M_i := W(m+i),
```

this is

```text
b^2 GR(k+1,m+1) GR(k+1,m) - HR(k+1,m) HR(k,m)
=
K_0 K_2 K_1^2 M_1^2 S(m)
+ K_1 M_2 T(k+1,m)
- K_0 K_2 M_0^2 M_1^2 S(k+1).
```

Here

```text
S(k+1) = W(k+3) W(k-1) - b^2 W(k+2) W(k) + c W(k+1)^2.
```

Cancellation factor for the generic H-step:

```text
V(k,m) = b W(m+k+1) W(m-k).
```

This factor vanishes in the universal sequence when `m = k` or `m = -k-1` (and also specializes if `b` maps to zero in a target ring).  In the universal-domain proof, split the vanishing-index cases.  For `m = k`, `OffRel(k+1,k)` is exactly the even normEDS recurrence

```text
b W(2k+2)
= W(k+3) W(k+1) W(k)^2
  - W(k-1) W(k+1) W(k+2)^2,
```

again up to the sign from `W(-1)=-1`.  The case `m = -k-1` follows by antisymmetry / the same even recurrence.

## 5. Base cases for GapRel

### Gap k = 0

```text
U(0,m) = W(m)^2.
B(0) = W(1)W(-1) = -1.
W(0)=0.

GR(0,m) = B(m) W(0)^2 - B(0) W(m)^2
        = 0 - (-1) W(m)^2
        = W(m)^2.

GapRes(0,m)=0.
```

### Gap k = 1

```text
U(1,m) = W(m+1)W(m-1) = B(m).
B(1) = W(2)W(0) = 0.
W(1)=1.

GR(1,m) = B(m) W(1)^2 - B(1) W(m)^2
        = B(m).

GapRes(1,m)=0.
```

### Gap k = 2

```text
U(2,m) = W(m+2)W(m-2).
W(2)=b.
B(2)=W(3)W(1)=c.

GR(2,m) = b^2 W(m+1)W(m-1) - c W(m)^2.

GapRes(2,m)
= W(m+2)W(m-2) - b^2 W(m+1)W(m-1) + c W(m)^2
= S(m).
```

So `GapRel(2,m)` is exactly adjacent Somos.

## 6. Base cases for OffRel

### Off k = 0

```text
V(0,m) = b W(m+1)W(m).

HR(0,m)
= W(m+2)W(m-1)W(1)W(0)
  - W(2)W(-1)W(m+1)W(m)
= 0 - b(-1)W(m+1)W(m)
= b W(m+1)W(m).

OffRes(0,m)=0.
```

### Off k = 1

```text
V(1,m) = b W(m+2)W(m-1).

HR(1,m)
= W(m+2)W(m-1)W(2)W(1)
  - W(3)W(0)W(m+1)W(m)
= b W(m+2)W(m-1).

OffRes(1,m)=0.
```

### Off k = 2

Using `W(4)=b d`, the residual is

```text
OffRes(2,m)
= b W(m+3)W(m-2)
  - (W(m+2)W(m-1)W(3)W(2) - W(4)W(1)W(m+1)W(m))
= b * O2core(m),
```

where

```text
O2core(m) := W(m+3)W(m-2)
             - c W(m+2)W(m-1)
             + d W(m+1)W(m).
```

This is not adjacent Somos alone.  The exact division-free certificate using your `Inv(m)` is:

```text
W(m-1) * O2core(m)
=
  W(m-2) * S(m+1)
+ b^2 W(m) * S(m)
- Inv(m).
```

Equivalently, for the actual `OffRes(2,m)` residual:

```text
W(m-1) * OffRes(2,m)
=
  b W(m-2) * S(m+1)
+ b^3 W(m) * S(m)
- b * Inv(m).
```

Expanded check of the core certificate:

```text
S(m)   = W(m+2)W(m-2) - b^2 W(m+1)W(m-1) + c W(m)^2,
S(m+1) = W(m+3)W(m-1) - b^2 W(m+2)W(m) + c W(m+1)^2,

Inv(m) = c( W(m+2)W(m-1)^2
          + W(m+1)^2 W(m-2)
          + b^2 W(m)^3 )
         - (d+b^4) W(m+1)W(m)W(m-1).
```

Indeed:

```text
W(m-1) O2core(m)
= W(m-2) S(m+1) + b^2 W(m) S(m) - Inv(m).
```

For `m = 1`, the cancellation factor `W(m-1)` is `W(0)=0`; prove directly from base values:

```text
V(2,1) = b W(4)W(-1) = -b^2 d,
HR(2,1) = W(3)W(0)W(3)W(2) - W(4)W(1)W(2)W(1)
        = 0 - (bd)(1)b(1)
        = -b^2 d.
```

So `OffRes(2,1)=0`.  For all `m ≠ 1`, the universal-domain proof cancels `W(m-1)` and then transports the resulting polynomial identity to the target commutative ring.

A symmetric alternative, useful if you prefer to avoid this exceptional split, is:

```text
W(m+2) * O2core(m)
=
  W(m+3) * S(m)
+ b^2 W(m+1) * S(m+1)
- Inv(m+1).
```

This follows by the same expansion.

## 7. Lean-facing summary

Use these residual constants in Lean:

```lean
-- schematic names only
GapRes k m := U k m - GR k m
OffRes k m := V k m - HR k m
SomosRes i := W (i+2) * W (i-2) - b^2 * W (i+1) * W (i-1) + c * W i^2
TRes i j := N i * D j - N j * D i
```

G-step linear combination:

```text
b^2 * U(k,m) * GapRes(k+1,m)
=
  1 * TauG(k,m)
+ V(k,m-1) * OffRes(k,m)
+ HR(k,m) * OffRes(k,m-1)
- b^2 * GR(k+1,m) * GapRes(k,m)
+ W(k)^2 * W(k+1)^2 * W(m+1) * W(m-1) * S(m)
+ W(m) * W(k+2) * T(k,m)
- W(k+1)^2 * W(m+1) * W(m)^2 * W(m-1) * S(k).
```

H-step linear combination:

```text
V(k,m) * OffRes(k+1,m)
=
  1 * TauH(k,m)
- HR(k+1,m) * OffRes(k,m)
+ b^2 * U(k+1,m+1) * GapRes(k+1,m)
+ b^2 * GR(k+1,m) * GapRes(k+1,m+1)
+ W(k) * W(k+2) * W(k+1)^2 * W(m+1)^2 * S(m)
+ W(k+1) * W(m+2) * T(k+1,m)
- W(k) * W(k+2) * W(m)^2 * W(m+1)^2 * S(k+1).
```

Base set:

```text
GapRes(0,m)=0               by W(0)=0, W(1)=1, W(-1)=-1
GapRes(1,m)=0               by W(1)=1, W(0)=0
GapRes(2,m)=S(m)            adjacent Somos

OffRes(0,m)=0               by W(0)=0, W(-1)=-1
OffRes(1,m)=0               by W(1)=1, W(0)=0, W(2)=b
OffRes(2,m)=0               by W(m-1)*OffRes(2,m)
                              = bW(m-2)S(m+1)+b^3W(m)S(m)-bInv(m),
                              plus direct m=1 case
```

The two main certificates above are the ones to feed to `linear_combination`; all signs are fixed by the residual orientations at the top of this file.
