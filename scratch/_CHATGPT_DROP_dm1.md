
for the short Weierstrass curve

```text
y² = x³ + A*x + B.
```

For `n = 6` and `n = 8`, the symbolic resultants are fast, but the extended Bézout cofactor extraction over `QQ(A,B)` did **not** finish in this environment. ## Definitions used

For the short Weierstrass specialization,

```text
Ψ₂Sq = 4*(X³ + A*X + B)
Ψ₃   = 3*X⁴ + 6*A*X² + 12*B*X - A²
preΨ₄ = 2*X⁶ + 10*A*X⁴ + 40*B*X³ - 10*A²*X² - 8*A*B*X - 2*A³ - 16*B²
```

and `preΨ'_n` is generated by the Mathlib recurrence

```text
preΨ' := preNormEDS' (Ψ₂Sq²) Ψ₃ preΨ₄.
```

Let

```text
D = 4*A^3 + 27*B^2,
Δ = -16*D.
```

## Complete runnable SymPy script

This script computes the reduced even polynomials, derivatives, resultants, resultant factor checks, and the `n=4` Bézout certificate.  The optional slow block attempts the `n=6,8` extended Bézout computations; it is disabled by default because it did not finish here.

```python
import sympy as sp
import time

X, A, B = sp.symbols('X A B')
D = 4*A**3 + 27*B**2
Delta = -16*D

Psi2Sq = 4*(X**3 + A*X + B)
Psi3 = 3*X**4 + 6*A*X**2 + 12*B*X - A**2
pre4 = 2*X**6 + 10*A*X**4 + 40*B*X**3 - 10*A**2*X**2 - 8*A*B*X - 2*A**3 - 16*B**2

pre = {
    0: sp.Integer(0),
    1: sp.Integer(1),
    2: sp.Integer(1),
    3: Psi3,
    4: pre4,
}

def prePsi(n: int):
    if n in pre:
        return pre[n]
    if n % 2 == 0:
        # n = 2*(m+3)
        m = n//2 - 3
        ans = prePsi(m+2)**2 * prePsi(m+3) * prePsi(m+5) - prePsi(m+1) * prePsi(m+3) * prePsi(m+4)**2
    else:
        # n = 2*(m+2)+1
        m = (n - 1)//2 - 2
        ans = (
            prePsi(m+4) * prePsi(m+2)**3 * (Psi2Sq**2 if m % 2 == 0 else 1)
            - prePsi(m+1) * prePsi(m+3)**3 * (1 if m % 2 == 0 else Psi2Sq**2)
        )
    pre[n] = sp.expand(ans)
    return pre[n]

# Precompute through 8.
for n in range(5, 9):
    prePsi(n)

expected = {
    4: (-2**29, 5),
    6: (2**176 * 3**13, 40),
    8: (-2**664, 145),
}

print('POLYNOMIAL SUMMARY')
for n in (4, 6, 8):
    f = prePsi(n)
    df = sp.diff(f, X)
    print(f'n={n}')
    print('  prePsi =', f)
    print('  dprePsi =', df)
    print('  deg =', sp.degree(f, X))
    print('  terms(prePsi) =', len(sp.Poly(f, X, A, B).terms()))
    print('  terms(dprePsi) =', len(sp.Poly(df, X, A, B).terms()))

print('\nRESULTANTS')
for n in (4, 6, 8):
    f = prePsi(n)
    df = sp.diff(f, X)
    t = time.time()
    Rn = sp.resultant(f, df, X)
    print(f'n={n}: resultant computed in {time.time() - t:.3f}s, terms={len(sp.Poly(Rn, A, B).terms())}')
    c, e = expected[n]
    assert sp.expand(Rn - c * D**e) == 0
    print(f'  Res = {c} * (4*A**3 + 27*B**2)**{e}')
    # Convert to Delta expression.
    delta_multiplier = sp.simplify(c / ((-16)**e))
    assert sp.expand(Rn - delta_multiplier * Delta**e) == 0
    print(f'  Res = {delta_multiplier} * Delta**{e}')

print('\nN=4 BEZOUT')
f4 = prePsi(4)
df4 = sp.diff(f4, X)
R4 = expected[4][0] * D**expected[4][1]
U4raw, V4raw, h = sp.gcdex(f4, df4, X)
assert h == 1
U4 = sp.expand(sp.cancel(U4raw * R4))
V4 = sp.expand(sp.cancel(V4raw * R4))
assert sp.denom(U4) == 1 and sp.denom(V4) == 1
assert sp.expand(U4*f4 + V4*df4 - R4) == 0
print('U4 terms =', len(sp.Poly(U4, X, A, B).terms()), 'deg_X =', sp.degree(U4, X))
print('V4 terms =', len(sp.Poly(V4, X, A, B).terms()), 'deg_X =', sp.degree(V4, X))
print('U4 =', U4)
print('V4 =', V4)

# Optional: this is expensive with SymPy.  It did not finish for n=6 in this environment.
TRY_SLOW_BEZOUT_6_8 = False
if TRY_SLOW_BEZOUT_6_8:
    for n in (6, 8):
        f = prePsi(n)
        df = sp.diff(f, X)
        c, e = expected[n]
        Rn = c * D**e
        print(f'Attempting gcdex for n={n} ...')
        t = time.time()
        Uraw, Vraw, h = sp.gcdex(f, df, X)
        print('  gcdex time =', time.time() - t)
        assert h == 1
        U = sp.expand(sp.cancel(Uraw * Rn))
        V = sp.expand(sp.cancel(Vraw * Rn))
        assert sp.denom(U) == 1 and sp.denom(V) == 1
        assert sp.expand(U*f + V*df - Rn) == 0
        print(f'  U{n} terms =', len(sp.Poly(U, X, A, B).terms()), 'deg_X =', sp.degree(U, X))
        print(f'  V{n} terms =', len(sp.Poly(V, X, A, B).terms()), 'deg_X =', sp.degree(V, X))

print('OK')
```
