# Q1071 (dm1): `dm3` — Kubert quartic obstruction index mapping

Using the convention for this obstruction family

```text
n = 2(d + 3) = 2d + 6,
```

the requested mappings are:

```text
d=2 → n=10
d=3 → n=12
d=4 → n=14
d=5 → n=16
d=6 → no n in {10,12,14,16}; under the same convention it would be n=18
d=7 → no n in {10,12,14,16}; under the same convention it would be n=20
```

So, restricted to `n ∈ {10,12,14,16}`, the active values are exactly

```text
2 ↦ 10, 3 ↦ 12, 4 ↦ 14, 5 ↦ 16.
```
